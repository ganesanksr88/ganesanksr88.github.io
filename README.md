# Ganesan Sankaran - Resume Website

A professional resume website built with Jekyll and hosted on GitHub Pages, showcasing my expertise in .NET development, Azure, and DevOps.

## Features

- Responsive design using Minimal Mistakes theme
- Static site generation with Jekyll
- SEO optimized
- Contact information and social links
- Experience, skills, education, and certifications pages
- Mobile-friendly layout

## Setup for Deployment

1. Create a new GitHub repository named `yourusername.github.io`
2. Clone the repository and replace with these files
3. Update `_config.yml` with your actual GitHub username
4. Push to GitHub main branch for automatic deployment to GitHub Pages

## Local Development

```bash
bundle install
bundle exec jekyll serve
```

Visit http://localhost:4000 to preview the site.

## Customization

- Update personal information in `_config.yml`
- Modify content in Markdown files (.md)
- Add an avatar image to `assets/images/avatar.jpg`
- Customize theme colors in theme settings
