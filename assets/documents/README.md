# Resume Documents

This folder contains downloadable resume files.

## Required Files

Please add your resume files with these exact names:

- `Ganesan_Sankaran.pdf` - Your resume in PDF format
- `Ganesan_Sankaran.docx` - Your resume in Microsoft Word format

These files are linked from the homepage download buttons and will be served directly from the repository.

## Notes

- PDF file size should be under 1MB for optimal loading
- DOCX file size should be under 500KB
- Both files should contain your most up-to-date resume information
- Update files whenever you make changes to your resume
